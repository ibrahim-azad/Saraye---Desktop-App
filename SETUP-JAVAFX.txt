                    JAVAFX SDK SETUP GUIDE - WINDOWS
================================================================================

Your application failed to run because JavaFX needs NATIVE DLL files.
You currently only have JAR files.

Follow these steps to fix it:

================================================================================
                         STEP 1: DOWNLOAD JAVAFX SDK
================================================================================

1. Open browser and go to: https://gluonhq.com/products/javafx/

2. Click the "Download" button

3. Select these options:
   - Version: JavaFX 21 (or latest LTS)
   - Operating System: Windows
   - Type: SDK

4. Download the ZIP file (approximately 50MB)
   Example: javafx-sdk-21.0.5-windows.zip


================================================================================
                         STEP 2: EXTRACT JAVAFX SDK
================================================================================

1. Extract the downloaded ZIP file to: C:\

2. Rename the folder to simply: C:\javafx-sdk-21.0.5
   (Remove any extra version numbers like -windows)

3. Verify the structure looks like this:

================================================================================
                STEP 3: UPDATE VS CODE SETTINGS (OPTIONAL)
================================================================================

Open .vscode/settings.json in your project and change:

FROM:
    "java.debug.settings.vmArgs": "--module-path \"CHANGE_THIS_PATH\" --add-modules javafx.controls,javafx.fxml",

TO:
    "java.debug.settings.vmArgs": "--module-path \"C:/javafx-sdk-21.0.5/lib\" --add-modules javafx.controls,javafx.fxml",

Save the file.


================================================================================
                STEP 4: UPDATE RUN SCRIPT (IF NEEDED)
================================================================================

The run.bat file has been updated to automatically look for JavaFX at:
C:\javafx-sdk-21.0.5\lib

If you extracted JavaFX to a different location, edit run.bat:

1. Open run.bat in notepad
2. Find this line:
   set JAVAFX_PATH=C:\javafx-sdk-21.0.5\lib
3. Change it to YOUR JavaFX location
4. Save the file

================================================================================
                            TROUBLESHOOTING
================================================================================

Problem: "JavaFX SDK not found"
Solution:
  - Make sure you extracted to C:\javafx-sdk-21.0.5\
  - Check the bin\ and lib\ folders exist inside
  - Update the path in run.bat if different

Problem: Still getting graphics errors
Solution:
  - Make sure you downloaded "SDK" not "jmods"
  - Make sure it's the Windows version
  - Try restarting VS Code after setup

Problem: "Cannot find module javafx.controls"
Solution:
  - The path should point to the lib folder: C:\javafx-sdk-21.0.5\lib
  - NOT the bin folder
  - NOT the root folder

Problem: Missing MSVCP140.dll or VCRUNTIME140.dll
Solution:
  - Download and install Visual C++ Redistributable from:
    https://aka.ms/vs/17/release/vc_redist.x64.exe
