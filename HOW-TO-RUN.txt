METHOD 1: Compile and Run (One Step)
-------------------------------------
1. Double-click: compile-and-run.bat
2. Wait for compilation to complete
3. Application window will open automatically


METHOD 2: Separate Compile and Run
-----------------------------------
1. Double-click: compile.bat     (compiles the code)
2. Double-click: run.bat          (runs the application)


METHOD 3: From VS Code
----------------------
1. Open project in VS Code
2. Press F5 or click "Run" -> "Start Debugging"
3. Application will launch

================================================================================
                         TROUBLESHOOTING
================================================================================

Problem: "javac is not recognized"
Solution: Make sure Java JDK is installed and added to PATH

Problem: Application doesn't start
Solution:
  1. Run compile.bat first
  2. Make sure JavaFX libraries are in lib/ folder
  3. Check if Java version is 11 or higher

Problem: CSS not loading
Solution: Resources are automatically copied during compilation

Problem: FXML not found
Solution: Make sure bin/ui/views/ contains .fxml files
